#Introduction
**Discord bot for tracking Savage World style bennies.**
This is a DiscordBot that can be used to track Savage World bennies.

## Help & Syntax
All commands starts with '!'.

## Adding
**Only usable by users with the 'GM' role**
To add a single bennie to a specified player: 
```
!add {player name}
```

To add multiple bennies to a specified player: 
```
!add {player name} {# of bennies}
```

## Using Bennies
To use a single bennie use the bennie command
```
!bennie
```

To use multiple bennies:
```
!bennie {# of bennies}
```
